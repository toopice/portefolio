Symfony - Site e-commerce

Dans le cadre de ce projet Symfony (v7.2), j'ai développé un site e-commerce à partir d'un squelette fourni. Ce travail m'a permis de renforcer ma compréhension du framework Symfony, notamment en ce qui concerne les routes, les contrôleurs, le système de templates Twig et la structuration d'un projet web MVC. Voici les principales réalisations :

  • Structure du site : mise en place d'un layout principal base.html.twig avec des blocs personnalisables (titre, contenu, scripts, styles), et inclusion d'une barre de navigation modulaire via navbar.html.twig.
  • Pages statiques et dynamiques : création d'une page de contact accessible via une route dédiée (/contact), avec un template Twig spécifique.
  • Affichage dynamique : utilisation d'un service Symfony (BoutiqueService) simulant une base de données via des tableaux JSON en dur pour afficher les catégories et produits.
  • Routage dynamique : mise en place de routes pour afficher les catégories (/boutique) et les produits associés (/rayon/{idCategorie}) via un contrôleur spécifique (BoutiqueController).
  • Templates Twig : développement des vues index.html.twig et rayon.html.twig pour afficher dynamiquement les contenus, avec intégration fluide des liens internes.
  • Connexion à une base de données SQL : intégration d'une base SQL pour stocker les informations sur les produits et les catégories, en utilisant Doctrine ORM pour gérer les entités et leurs relations.
  • Gestion des utilisateurs : mise en place d'un système d'inscription et de connexion pour permettre aux utilisateurs de créer un compte, gérer leur panier et suivre leurs commandes. Les données sont persistées en base.

Ce TP a été très formateur et m'a permis de maîtriser les fondements du développement web avec Symfony tout en appliquant les bonnes pratiques de modularité, de routage, de templating et de gestion de données.
